# Scrimba HTML & CSS Solo Projects
This repository contains solo projects completed as part of the Scrimba learn HTML and CSS course. Each folder contains one project with HTML and CSS files.

## Projects
1. **Amazon Product Page**  
   A simple replica of an Amazon product listing showcasing layout structure and image alignment.

2. **Hometown Homepage**  
   A personal homepage highlighting a hometown, using flexbox layout.

3. **The Daily Dribble Newsletter**  
   A clean, centered newsletter join now page.

4. **Web Agency Hero**  
   A modern hero section for a fictional web agency
